Crie um script que receba o preço original de um produto e a categoria do cliente (como "comum", "premium" ou "vip"). O script deve calcular o valor do desconto com base na categoria e printar o preço final. As regras de desconto são as seguintes:

Clientes "comuns" têm 5% de desconto.
Clientes "premium" têm 10% de desconto.
Clientes "vip" têm 15% de desconto.
Se a categoria não for reconhecida, o script deve printar o preço original sem desconto.
