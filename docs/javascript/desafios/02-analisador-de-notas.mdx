Crie um script simples que analisa as notas dos alunos de uma sala de aula.

O script precisa:
- Exibir a média das notas de cada aluno.
- Exibir quantos alunos ficaram acima e abaixo da média. `média = 6`
- Exibir a maior e a menor nota da turma.

A estrutura de dados que usaremos:
```js
const alunos = [
  { nome: "João Paulo", nota1: 7, nota2: 5, nota3: 8 },
  { nome: "Maria Clara", nota1: 9, nota2: 6, nota3: 7 },
  { nome: "Carlos Eduardo", nota1: 4, nota2: 5, nota3: 6 },
  { nome: "Ana Beatriz", nota1: 10, nota2: 9, nota3: 8 },
  { nome: "Pedro Henrique", nota1: 5, nota2: 6, nota3: 5 },
];

// Considere que as notas são valores numéricos entre 0 e 10.
```
