## O que é JavaScript?

**JavaScript** é uma linguagem de programação de alto nível, interpretada, que se tornou uma das principais linguagens para o desenvolvimento web. Criada em 1995, inicialmente servia para adicionar interatividade a sites, mas hoje é usada tanto no front-end quanto no back-end (Node.js).

### Qual sua Importância?

- No **Desenvolvimento Web** a maior parte dos sites utiliza JavaScript para dinamizar as interfaces e melhorar a experiência do usuário.
- O **JS** é usando vários ecossistemas como o desenvolvimento mobile (React Native), nos servidores (Node.js), e até em desktop apps (Electron).

<Admonition type="info" title="Diferença entre JavaScript e outras linguagens">
  <Tabs>
    <TabItem value="java" label="Java">
      - Ambos têm nomes semelhantes, mas são linguagens bem diferentes. **Java**
      é uma linguagem fortemente tipada, compilada, enquanto o **JavaScript** é
      dinamicamente tipado e interpretado. - **Java** geralmente é utilizado
      para desenvolvimento de aplicativos mais complexos, como sistemas backend
      robustos, enquanto **JavaScript** é mais focado em interatividade na web,
      mas ganhou espaço no back-end com Node.js.
    </TabItem>
    <TabItem value="python" label="Python">
      - Python é conhecido pela simplicidade e é amplamente utilizado em ciência
      de dados, automação e desenvolvimento web (com frameworks como
      Django/Flask). JavaScript, por outro lado, tem como foco inicial o
      desenvolvimento web, mas também tem sido utilizado em outras áreas pela
      sua praticidade e flexibilidade
    </TabItem>
    <TabItem value="c" label="C">
      - C é uma linguagem de baixo nível, usada para desenvolvimento de sistemas
      e software de performance crítica. JavaScript é de alto nível e é mais
      acessível para desenvolvimento rápido de aplicações web.
    </TabItem>
  </Tabs>
</Admonition>
