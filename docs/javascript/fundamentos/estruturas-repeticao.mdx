## Explicação Conceitual

As estruturas de repetição, ou loops, são ferramentas essenciais em programação que permitem executar um bloco de código múltiplas vezes. Em JavaScript, temos três principais tipos de loops:

## For Loop

O for loop é usado quando sabemos antecipadamente quantas vezes queremos que o código seja executado.

### Sintaxe

<Example language="js" filename="for-loop-sintaxe.js" />

### Explicação:

- **Inicialização**: Executada uma vez no início do loop. Geralmente usada para definir uma variável contadora.
- **Condição**: Verificada antes de cada iteração. Se for verdadeira, o loop continua; se falsa, o loop termina.
- **Expressão** final: Executada no final de cada iteração, geralmente para atualizar o contador.

### Exemplo

<Example language="js" filename="for-loop-exemplo.js" />

## While Loop

O while loop é usado quando queremos que o código seja executado enquanto uma condição específica for verdadeira.

### Sintaxe

<Example language="js" filename="while-loop-sintaxe.js" />

### Explicação:

- A **condição** é verificada antes de cada iteração.
- Se a condição for verdadeira, o código dentro do loop é executado.
- Se a condição for falsa, o loop termina.

### Exemplo

<Example language="js" filename="while-loop-exemplo.js" />

## Do...While Loop

O do...while loop é similar ao while, mas garante que o código seja executado pelo menos uma vez, pois a condição é verificada no final da iteração.

### Sintaxe

<Example language="js" filename="do-while-loop-sintaxe.js" />

### Explicação:

- A **condição** é verificada antes de cada iteração.
- Se a condição for verdadeira, o código dentro do loop é executado.
- Se a condição for falsa, o loop termina.

<Admonition type="info" title="Considerações Importantes">
  - **Evitando Loops Infinitos**: Sempre certifique-se de que a condição do loop
  eventualmente se torne falsa, ou use break para sair do loop. - **Break e
  Continue**: break: Sai imediatamente do loop. continue: Pula para a próxima
  iteração do loop. - **Escolhendo o Loop Certo**: Use for quando souber o
  número exato de iterações. Use while quando a condição de parada não for
  baseada em um contador simples. Use do...while quando precisar que o código
  seja executado pelo menos uma vez. - **Performance**: Em geral, for loops são
  ligeiramente mais rápidos, mas a diferença é negligenciável na maioria dos
  casos. Loops Aninhados: É possível ter loops dentro de loops, mas cuidado com
  a complexidade e o desempenho.
</Admonition>


## Exercício Propostos

### Exercício 1: Soma de Números com For Loop
Use um `for` loop para calcular e exibir a soma dos números de 1 a 100.

<details>
  <summary>Resposta</summary>
```js
let soma = 0; // Inicializa a variável soma com 0

// Loop que vai de 1 a 100
for (let i = 1; i <= 100; i++) {
  soma += i; // Adiciona o valor de i à soma em cada iteração
}

console.log("Soma de 1 a 100:", soma); // Exibe o resultado final
```
</details>

### Exercício 2: Contagem Regressiva com For Loop
Crie um loop `for` que exiba uma contagem regressiva de 10 até 1 no console.
<details>
  <summary>Resposta</summary>
```js
// Loop que começa em 10 e vai até 1
for (let i = 10; i >= 1; i--) {
  console.log(i); // Exibe o valor de i a cada iteração
}
```
</details>
### Exercício 3: Jogo de Adivinhação com While Loop
Usando um loop `while`, crie um programa que peça ao usuário para adivinhar um número entre 1 e 10. O loop deve continuar até que o usuário acerte o número correto.
<details>
  <summary>Resposta</summary>
```js
const numeroCorreto = Math.floor(Math.random() * 10) + 1; // Gera um número aleatório entre 1 e 10
let tentativa; // Declara a variável tentativa

// Loop continua até que a tentativa seja igual ao número correto
while (tentativa !== numeroCorreto) {
  tentativa = parseInt(prompt("Adivinhe um número entre 1 e 10:")); // Pede ao usuário um número

  // Verifica se a tentativa é igual ao número correto
  if (tentativa === numeroCorreto) {
    console.log("Parabéns! Você acertou.");
  } else {
    console.log("Tente novamente."); // Exibe mensagem para tentar de novo
  }
}
```
</details>
### Exercício 4: Menu com Do...While Loop
Crie um programa que exiba um menu com opções e use um loop `do...while` para perguntar ao usuário qual opção ele quer selecionar. O programa deve terminar apenas quando o usuário escolher a opção "Sair".
<details>
  <summary>Resposta</summary>
```js
let opcao; // Declara a variável opcao

// Loop exibe o menu e espera uma opção até o usuário escolher sair
do {
  console.log("Menu:\n1. Opção 1\n2. Opção 2\n3. Sair"); // Exibe o menu
  opcao = parseInt(prompt("Escolha uma opção:")); // Pede ao usuário uma opção

  // Verifica a opção selecionada
  switch (opcao) {
    case 1:
      console.log("Você escolheu a Opção 1.");
      break;
    case 2:
      console.log("Você escolheu a Opção 2.");
      break;
    case 3:
      console.log("Saindo...");
      break;
    default:
      console.log("Opção inválida."); // Mensagem de erro para opções inválidas
  }
} while (opcao !== 3); // O loop para apenas quando a opção "3" é selecionada

```
</details>
### Exercício 5: Fatorial de um Número com For Loop
Usando um loop `for`, calcule o fatorial de um número inteiro fornecido pelo usuário.
<details>
  <summary>Resposta</summary>
```js
const numero = parseInt(prompt("Digite um número para calcular o fatorial:")); // Pede ao usuário um número
let fatorial = 1; // Inicializa o fatorial como 1

// Loop para calcular o fatorial multiplicando de 1 até o número fornecido
for (let i = 1; i <= numero; i++) {
  fatorial *= i; // Multiplica o valor atual de fatorial por i
}

console.log(`Fatorial de ${numero}:`, fatorial); // Exibe o fatorial calculado
```
</details>
### Exercício 6: Números Primos com For Loop
Usando um `for` loop, exiba todos os números primos entre 1 e 50.
<details>
  <summary>Resposta</summary>
```js
for (let num = 2; num <= 50; num++) {
  let ehPrimo = true; // Assume que o número é primo

  // Verifica se o número é divisível por qualquer número de 2 até a raiz quadrada dele
  for (let divisor = 2; divisor <= Math.sqrt(num); divisor++) {
    if (num % divisor === 0) {
      ehPrimo = false; // Se for divisível, não é primo
      break; // Sai do loop de divisor
    }
  }

  // Se ehPrimo ainda for true, o número é primo
  if (ehPrimo) {
    console.log(num); // Exibe o número primo
  }
}
```
</details>
