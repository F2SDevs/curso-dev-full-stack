## Explicação Conceitual

Conversão de dados, ou casting, é o processo de transformar um tipo de dado em outro. Em JavaScript, podemos realizar conversões de forma explícita ou implícita. Existem três tipos principais de conversões: para número, para string e para booleano.

## Conversão para Número

Quando precisamos transformar outros tipos de dados em números, podemos usar métodos específicos ou confiar em conversões implícitas feitas pelo JavaScript.

### Sintaxe

<Example language="js" filename="conversao-para-numero-sintaxe.js" />

### Explicação:

- **parseInt()**: Converte uma string em um número inteiro. Ignora caracteres não numéricos após os dígitos iniciais.
- **parseFloat()**: Converte uma string em um número de ponto flutuante.
- **Number()**: Converte strings e outros tipos de dados diretamente em um número, incluindo NaN se a conversão falhar.

- **String()**: Converte qualquer valor diretamente para uma string.
- **toString()**: Método chamado em variáveis numéricas, booleanas ou objetos para convertê-los em strings.

## Conversão para Booleano

Em JavaScript, valores podem ser convertidos para booleanos, onde qualquer valor pode ser avaliado como `true` ou `false`.

### Valores que são **false**:

- `false`
- `0`
- `""` (string vazia)
- `null`
- `undefined`
- `NaN`

### Exemplo

<Example language="js" filename="conversao-para-booleano-exemplo.js" />

### Explicação:

A conversão booleana ocorre frequentemente em condições (ex: em estruturas de controle como `if`). O JavaScript automaticamente converte o valor para um booleano, onde certos valores são considerados falsos (falsy) e todos os outros são considerados verdadeiros (truthy).

## Conversão Implícita

O JavaScript pode realizar conversões automáticas, conhecidas como coerção de tipos, quando tenta operar com diferentes tipos de dados.

### Exemplo

<Example language="js" filename="conversao-implicita-exemplo.js" />

### Explicação:

- Quando usamos operadores como `+` com strings e números, o JavaScript pode converter automaticamente os números em strings.
- Comparações como `==` também podem realizar conversões implícitas, ao contrário de `===`, que não permite coerção de tipos.

import Admonition from "@theme/Admonition";

<Admonition type="info" title="Considerações Importantes">
  - **parseInt vs Number**: `parseInt` ignora caracteres não numéricos após os
  primeiros dígitos, enquanto `Number` tenta converter a string inteira. 
  - **Evitando Coerção de Tipos**:
   Para evitar resultados inesperados, prefira
  sempre usar `===` ao invés de `==` nas comparações. 
  - **Conversão Automática**: 
  A conversão automática de tipos pode causar comportamentos
  estranhos, como `"5" + 1` resultando em `"51"`. 
  - **Conversão Explícita**:
   Para evitar erros, é uma boa prática usar métodos de conversão explícitos como
  `Number()`, `String()` ou `Boolean()` quando necessário.
</Admonition>



## Exercícios Propostos

### Exercício 1: Convertendo String para Número
Declare uma variável `idadeString` como uma string que representa um número, por exemplo, `"25"`. Em seguida, converta essa string para um número usando `parseInt`, `parseFloat` e `Number`, e exiba os resultados. Observe a diferença entre os métodos.

<details>
  <summary>Resposta</summary>
```js
const idadeString = "25";

const idadeInt = parseInt(idadeString); // Converte para inteiro
const idadeFloat = parseFloat(idadeString); // Converte para ponto flutuante
const idadeNumber = Number(idadeString); // Converte usando Number()

console.log(idadeInt);    // 25
console.log(idadeFloat);  // 25
console.log(idadeNumber); // 25
</details>
```
</details>

### Exercício 2: Convertendo Número para String
Crie uma variável `preco` com um valor numérico, por exemplo, `49.99`. Em seguida, converta esse número para uma string usando `String()` e `toString()` e exiba os resultados. Observe como o valor é exibido como uma string.

<details>
  <summary>Resposta</summary>
```js
const preco = 49.99;

const precoString1 = String(preco);   // Converte usando String()
const precoString2 = preco.toString(); // Converte usando toString()

console.log(precoString1); // "49.99"
console.log(precoString2); // "49.99"
```
</details>

---

### Exercício 3: Convertendo Valores para Booleanos
Crie uma variável `valor` e atribua valores diferentes a ela, como `0`, `1`, `""`, `"hello"`, `null`, e `undefined`. Use `Boolean()` para converter cada um desses valores em um booleano e exiba o resultado. Quais valores são `true` e quais são `false`?

<details>
  <summary>Resposta</summary>
```js
const valor1 = 0;
const valor2 = 1;
const valor3 = "";
const valor4 = "hello";
const valor5 = null;
const valor6 = undefined;

console.log("0:", Boolean(valor1));          // false
console.log("1:", Boolean(valor2));          // true
console.log('"" (string vazia):', Boolean(valor3)); // false
console.log('"hello":', Boolean(valor4));    // true
console.log("null:", Boolean(valor5));       // false
console.log("undefined:", Boolean(valor6));  // false
```
</details>

### Exercício 4: Coerção de Tipos em Operações
Declare uma variável `numero` com o valor `5` e outra variável `texto` com o valor `"10"`. Use o operador `+` para somar `numero` e `texto`, e observe o resultado. Em seguida, use `Number()` para converter `texto` antes da soma e exiba o resultado. Analise a diferença.

<details>
  <summary>Resposta</summary>
```js
const numero = 5;
const texto = "10";

const resultadoComCoercao = numero + texto; // Coerção implícita para string
console.log(resultadoComCoercao); // "510"

const resultadoSemCoercao = numero + Number(texto); // Conversão explícita
console.log(resultadoSemCoercao); // 15
```
</details>

---

### Exercício 5: Comparação com Conversão Implícita
Declare duas variáveis: `a` com o valor `5` (número) e `b` com o valor `"5"` (string). Compare `a` e `b` usando `==` e `===`, e exiba o resultado de cada comparação. Analise qual a diferença entre os dois operadores.

<details>
  <summary>Resposta</summary>
```js
const a = 5;       // Número
const b = "5";     // String

console.log(a == b);  // true (coerção de tipo)
console.log(a === b); // false (sem coerção de tipo)
```
</details>

<Admonition type="tip" title="Dica">
- **Seja cauteloso com a coerção de tipos**: Coerções automáticas podem ser úteis, mas também podem levar a resultados inesperados, como `"5" + 2` resultando em `"52"`. Prefira conversões explícitas quando possível.
</Admonition>
