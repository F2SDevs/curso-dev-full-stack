## If Else

### If

A estrutura `if` avalia uma condição e executa um bloco de código se a condição for `true` (verdadeira).

<Admonition>
  A condição não precisa ser necessariamente `true`. Ela pode ser qualquer valor que convertido para `boolean` de `true`
</Admonition>

**Exemplo**

```js
const idade = 18;

if (idade >= 18) {
    console.log('Você é maior de idade.');
}
```
Neste exemplo, o código dentro do bloco if será executado somente se a variável idade for maior ou igual a 18.

### else

A estrutura `else` permite executar um bloco de código caso a condição seja verdadeira, e outro bloco diferente caso seja falsa.

**Exemplo**

```js
const idade = 16;

if (idade >= 18) {
    console.log('Você é maior de idade.');
} else {
    console.log('Você é menor de idade.');
}

```
Aqui, se a condição não for satisfeita (ou seja, se a idade for menor que 18), o código dentro do bloco `else` será executado.

### else if

A estrutura `else if` é utilizada quando há múltiplas condições a serem verificadas em sequência. Caso uma condição seja verdadeira, seu bloco correspondente é executado, e as outras condições são ignoradas.

**Exemplo**

```js
const hora = 17;

if (hora < 12) {
    console.log('Bom dia!');
} else if (hora < 18) {
    console.log('Boa tarde!');
} else {
    console.log('Boa noite!');
}

```
Dependendo do valor da variável hora, o código exibirá a mensagem apropriada.

## Operador Ternário

O `operador ternário` é uma maneira mais concisa de escrever uma expressão condicional. Ele avalia uma condição e retorna um valor se a condição for true e outro valor se for false.

**Exemplo**

```js
const idade = 20;
const status = idade >= 18 ? 'maior de idade' : 'menor de idade';

console.log(`Você é ${status}.`);
```
Aqui, a variável status recebe o valor 'maior de idade' ou 'menor de idade', dependendo do valor da variável idade. O operador ternário simplifica a escrita de condicionais simples em uma única linha.

## Switch

A estrutura `switch` é usada quando há múltiplos valores possíveis para uma variável e você deseja executar diferentes blocos de código dependendo do valor.

**Exemplo**

```js
const diaDaSemana = 3;

switch (diaDaSemana) {
    case 1:
        console.log('Domingo');
        break;
    case 2:
        console.log('Segunda-feira');
        break;
    case 3:
        console.log('Terça-feira');
        break;
    case 4:
        console.log('Quarta-feira');
        break;
    case 5:
        console.log('Quinta-feira');
        break;
    case 6:
        console.log('Sexta-feira');
        break;
    case 7:
        console.log('Sábado');
        break;
    default:
        console.log('Dia inválido');
}

```
Neste exemplo, dependendo do valor de `diaDaSemana`, um dos blocos será executado. O `default` é executado caso nenhum dos valores correspondentes seja encontrado.
