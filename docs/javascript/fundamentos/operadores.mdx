## Explicação Conceitual

Operadores são símbolos ou palavras-chave que permitem realizar operações em variáveis e valores. Em JavaScript, temos diversos tipos de operadores, como aritméticos, relacionais, lógicos e de atribuição.

## Operadores Aritméticos


Os operadores aritméticos são usados para executar operações matemáticas básicas, como adição, subtração, multiplicação, divisão e potenciação (representada por "**"). Além disso, outros operadores como módulo (resto da divisão), incremento e decremento, que não são tão comuns no dia a dia, serão detalhados.

#### Módulo ou Resto da Divisão (%):

O operador ``%`` mostra quanto sobra quando dividimos um número por outro. Por exemplo ``5 % 2 = 1``, porque se eu dividir 5 por 2 sobra 1 (porque "2" cabe 2 vezes em 5, e sobra 1).


#### Incremento(++):

O operador "++" aumenta o valor de uma variável em 1.
Por exemplo, se você tem ``let idade = 1``; e depois coloca ``idade++``, o valor de idade se torna 2. É como dizer: "Variável, aumente esse número em 1!"


#### Decremento (--):

O operador "--" segue a mesma lógica do de incremento, só que nesse caso subtraíndo. Ele diminui o valor de uma variável em 1. Exemplo, se você tem let ``peso = 60``; e depois coloca ``peso--``, o valor de a se torna 59. É como dizer: "Variável, diminua esse número em 1!"

### Tabela de Operadores Aritméticos

| Operador | Descrição        | Exemplo      |
|----------|------------------|--------------|
| `+`      | Adição           | `5 + 2`      |
| `-`      | Subtração        | `5 - 2`      |
| `*`      | Multiplicação     | `5 * 2`      |
| `**`      | Potenciação    | `5 ** 2`      |
| `/`      | Divisão          | `5 / 2`      |
| `%`      | Módulo (Resto da divisão)   | `5 % 2`      |
| `++`     | Incremento       | `let a = 1; a++` |
| `--`     | Decremento       | `let a = 1; a--` |



### Exemplo

```js
// Adição
let soma = 5 + 2;
console.log("Adição: 5 + 2 =", soma); // 7

// Subtração
let subtracao = 5 - 2;
console.log("Subtração: 5 - 2 =", subtracao); // 3

// Multiplicação
let multiplicacao = 5 * 2;
console.log("Multiplicação: 5 * 2 =", multiplicacao); // 10

// Potenciação
let potenciacao = 5 ** 2;
console.log("Potenciação: 5 ** 2 =", potenciacao); // 25

// Divisão
let divisao = 5 / 2;
console.log("Divisão: 5 / 2 =", divisao); // 2.5

// Módulo (Resto da divisão)
let modulo = 5 % 2;
console.log("Módulo: 5 % 2 =", modulo); // 1

// Incremento
let idade = 1;
idade++; // idade = idade + 1
console.log("Incremento: let a = 1; a++ =", idade); // 2

// Decremento
let peso = 1;
peso--; // peso = peso - 1
console.log("Decremento: let b = 1; b-- =", peso); // 59
```

## Operadores Relacionais

Os operadores relacionais comparam valores e retornam um valor booleano (verdadeiro ou falso). Por exemplo: 5 `>` 3 = true. Nesse caso o `>`(maior que) é o operador relacional. Segue a tablea de operadores relacionais.

### Tabela de Operadores Relacionais

| Operador | Descrição                  | Exemplo         |
|----------|----------------------------|-----------------|
| `==`     | Igual a                    | `5 == 5`        |
| `===`    | Estritamente igual a       | `5 === 5`       |
| `!=`     | Diferente de               | `5 != 2`        |
| `!==`    | Estritamente diferente de  | `5 !== 2`       |
| `>`      | Maior que                  | `5 > 2`         |
| `<`      | Menor que                  | `2 < 5`         |
| `>=`     | Maior ou igual a           | `5 >= 2`        |
| `<=`     | Menor ou igual a           | `2 <= 5`        |

<Admonition type="note" title="Nota">
- **Maior que** `(>)`: Verifica se o valor à esquerda é maior que o valor à direita.
Exemplo: 5 `>` 3 → true

- **Menor que** ``<``: Verifica se o valor à esquerda é menor que o valor à direita.
Exemplo: 2 `<` 4 → true

- **Maior ou igual a** `(>=)`: Verifica se o valor à esquerda é maior ou igual ao valor à direita.
Exemplo: 5 `>=` 5 → true

- **Menor ou igual a** `(<=)`: Verifica se o valor à esquerda é menor ou igual ao valor à direita.
Exemplo: 3 `<=` 3  true

- **Igual a** `(==)`: Compara se os valores dos dois operandos são iguais (faz a conversão de tipos, se necessário).
Exemplo: 5 `==` '5' → true (porque há conversão de tipos)

- **Estritamente igual a** `(===)`: Compara se os valores e tipos dos dois operandos são iguais (não faz conversão de tipos).
Exemplo: 5 `===` '5' → false (porque os tipos são diferentes)

- **Diferente de** `(!=)`: Verifica se os valores dos dois operandos são diferentes (faz a conversão de tipos, se necessário).
Exemplo: 5 `!=` '5' → false (porque os valores são considerados iguais)

- **Estritamente diferente de** `(!==)`: Verifica se os valores ou tipos dos dois operandos são diferentes (não faz conversão de tipos).
Exemplo: 5 `!==` '5' → true (porque os tipos são diferentes)

</Admonition>

### Exemplo

```js
// Igual a
console.log(5 == 5); // true

// Estritamente igual a
console.log(5 === 5); // true

// Diferente de
console.log(5 != 2); // true

// Estritamente diferente de
console.log(5 !== 2); // true

// Maior que
console.log(5 > 2); // true

// Menor que
console.log(2 < 5); // true

// Maior ou igual a
console.log(5 >= 2); // true

// Menor ou igual a
console.log(2 <= 5); // true
```


## Operadores Lógicos

Os operadores lógicos são usados para combinar expressões condicionais.

- Operador && (E - AND)
Descrição: O operador && é usado para combinar duas expressões booleanas. Ele retorna true apenas se ambas as expressões forem verdadeiras.
Comportamento:
Se o primeiro operando for false, o resultado será false (não precisa avaliar o segundo operando).
Se o primeiro operando for true, o resultado dependerá do segundo operando.

- Operador || (Ou - OR)
Descrição: O operador || é usado para combinar duas expressões booleanas. Ele retorna true se pelo menos uma das expressões for verdadeira.
Comportamento:
Se o primeiro operando for true, o resultado será true (não precisa avaliar o segundo operando).
Se o primeiro operando for false, o resultado dependerá do segundo operando.

- Operador ! (Não - NOT)
Descrição: O operador ! é um operador unário que inverte o valor booleano de uma expressão. Se a expressão for true, ele retorna false, e vice-versa.
Comportamento:
O operador ! sempre retornará o oposto do valor original.

### Tabela de Operadores Lógicos

| Operador | Descrição               | Exemplo            |
|----------|-------------------------|--------------------|
| `&&`     | E (AND)           | `true && false`    |
| `||`     | Ou (OR)           | `true || false`    |
| `!`      | Não (NOT)         | `!true`            |

### Exemplo

```js
// Exemplos de operadores lógicos

// E (AND)
let a = true;
let b = false;
console.log(a && b); // false

// Ou (OR)
let c = true;
let d = false;
console.log(c || d); // true

// Não (NOT)
let e = true;
console.log(!e); // false
```

## Operadores de Atribuição


Os operadores de atribuição são usados em programação para atribuir valores a variáveis. Eles simplificam a forma como você define ou atualiza o valor de uma variável. O operador de atribuição mais comum é o sinal de igual (=), mas existem vários operadores de atribuição que combinam a atribuição com outras operações aritméticas ou lógicas.


- = (Atribuição Simples):
Atribui o valor do operando da direita à variável à esquerda.

- += (Atribuição com Adição):
Adiciona o valor do operando da direita à variável à esquerda e atribui o resultado à variável. 

- -= (Atribuição com Subtração):
Subtrai o valor do operando da direita da variável à esquerda e atribui o resultado à variável.

- *= (Atribuição com Multiplicação):
Multiplica a variável à esquerda pelo valor do operando da direita e atribui o resultado à variável.

- /= (Atribuição com Divisão):
Divide a variável à esquerda pelo valor do operando da direita e atribui o resultado à variável.

- %= (Atribuição com Módulo):
Aplica o operador módulo (resto da divisão) e atribui o resultado à variável.

### Tabela de Operadores de Atribuição

| Operador | Descrição                         | Exemplo           |
|----------|-----------------------------------|-------------------|
| `=`      | Atribuição                        | `x = 5`           |
| `+=`     | Atribuição de adição              | `x += 5`          |
| `-=`     | Atribuição de subtração           | `x -= 5`          |
| `*=`     | Atribuição de multiplicação       | `x *= 5`          |
| `/=`     | Atribuição de divisão             | `x /= 5`          |
| `%=`     | Atribuição de módulo              | `x %= 5`          |

### Exemplo

```js
// Atribuição
let x = 5; // x agora é 5

// Atribuição de adição
x += 5; // x agora é 10 (5 + 5)

// Atribuição de subtração
x -= 3; // x agora é 7 (10 - 3)

// Atribuição de multiplicação
x *= 2; // x agora é 14 (7 * 2)

// Atribuição de divisão
x /= 7; // x agora é 2 (14 / 7)

// Atribuição de módulo
x %= 2; // x agora é 0 (resto de 2 dividido por 2)
```

 **A ordem de precedência de operadores** 

 Assim como na matemática, em Javascript existe uma ordem de precedência de operadores. Tal precedência determina em que ordem os operadores são avaliados em expressões. Aqui está uma lista da ordem de precedência, do mais alto para o mais baixo:

#### Operadores Aritméticos
- Ordem de precedência `** > * / % > + -`

#### Operadores relacionais
- Ordem de precedência `> < >= <=`

#### Operadores lógicos:

- ***Ordem de precedência*** `! > && > ||`

#### Atribuição:

- ***Ordem de precedência*** `= += -= *= /= %= <<= >>= >>>=`

#### Ordem geral de precedência dos operadores

- Operdaores aritimesticos> Operdaores relacionais> Operadores lógicos

<Admonition type="info" title="Considerações Importantes">
  - **Atribuição Combinada**: Operadores de atribuição como `+=` são atalhos que combinam operação e atribuição em uma única expressão.
  - **Curto-circuito em Operadores Lógicos**: Operadores lógicos podem evitar a avaliação de parte de uma expressão se o resultado já puder ser determinado (por exemplo, `false && ...` sempre será `false`).
</Admonition>

## Exercícios Propostos

### Exercício 1: Operadores Aritméticos

Crie duas variáveis, `a` e `b`, e atribua valores a elas. Em seguida, execute as seguintes operações: adição, subtração, multiplicação, divisão e módulo. Exiba o resultado de cada operação no console.

<details>
  <summary>Resposta</summary>

 

```js
let a = 10;  
let b = 3;  

console.log("Adição: ", a + b);  // Saída: 13
console.log("Subtração: ", a - b);  // Saída: 7
console.log("Multiplicação: ", a * b);   // Saída: 30
console.log("Divisão: ", a / b);  // Saída: 3.3333333333333335
console.log("Módulo: ", a % b);  // Saída: 1
```
</details>

### Exercício 2: Incremento e Decremento

Crie uma variável `contador` com o valor inicial de 5. Aplique o operador de incremento (`++`) e depois o operador de decremento (`--`) nela, e exiba o valor após cada operação.

<details>
  <summary>Resposta</summary>

```js
let contador = 5;  
contador++;  
console.log("Incremento: ", contador); // 6  
contador--;  
console.log("Decremento: ", contador); // 5  
```

</details>


### Exercício 3: Comparação de Valores

Crie duas variáveis, `x` e `y`, com valores diferentes. Utilize os operadores relacionais para comparar essas variáveis e exibir o resultado (verdadeiro ou falso) no console.

<details> 
  <summary>Resposta</summary>
```js
let x = 10;  
let y = 5;  

console.log(x > y); // true  
console.log(x < y); // false  
console.log(x == y); // false  
console.log(x != y); // true  
```
</details>

### Exercício 4: Operadores Lógicos
Crie duas variáveis booleanas, verdadeiro e falso, e use os operadores lógicos `&&`, `||` e `!` para combinar essas variáveis. Exiba o resultado de cada operação no console.

<details>
 <summary>Resposta</summary>
 ```js
let verdadeiro = true;  
let falso = false;  

console.log(verdadeiro && falso); // false  
console.log(verdadeiro || falso); // true  
console.log(!verdadeiro); // false  
```
</details>


### Exercício 5: Atribuições Compostas
Crie uma variável total com o valor inicial de `100`. Aplique operadores de atribuição composta (`+=`, `-=`) para adicionar e subtrair valores dessa variável, e exiba o resultado no console.

<details> 
  <summary>Resposta</summary>
```js
let total = 100;  
total += 50;  
console.log(total); // 150  
total -= 30;  
console.log(total); // 120  
```
</details>
