JavaScript é uma linguagem dinâmica, onde o tipo de dado de uma variável é determinado durante a execução. Aqui veremos os principais tipos de dados primitivos.

## String

Uma `string` é uma sequência de caracteres usada para representar texto. Strings podem ser delimitadas por aspas simples `'` ou duplas `"`.

### Sintaxe

```js
const saudacao = 'Olá, mundo!';
console.log(saudacao); // Saída: "Olá, mundo!"


const mensagem = "Como você está?";
console.log(saudacao); // Saída: "Como você está?"
```

### Concatenação

O que é Concatenar?
Concatenar significa juntar ou combinar dois ou mais valores em um único resultado. A concatenação de strings é o processo de unir várias strings em uma só. Essa operação é bastante comum quando precisamos combinar textos ou valores dinâmicos em uma única frase ou mensagem.

#### Há várias maneiras de concatenar strings:

- Usando o operador +.
- Usando template literals, é possível inserir variáveis e expressões diretamente dentro de uma string. Para isso, o texto é colocado entre crases (``), e `${}` é utilizado para inserir o valor de variáveis ou expressões dentro da string.
- Usando o método .concat(), que é um método de string que faz a concatenação.

```js
const saudacao = "Olá";
const nome = "Kevin";

// Concatenando usando o operador +
const resultado1 = saudacao + nome;
console.log(resultado1); // Saída: "Olá, Kevin"

// Concatenando usando o operador +
const resultado2 = saudacao + nome + ", como vai?";
console.log(resultado1); // Saída: "Olá, Kevin, como vai?"

// Concatenando usando template literals
const resultado3 = `${saudacao}, ${nome}, como vai?`;
console.log(resultado2); // Saída: "Olá, Kevin, como vai?"

// Concatenando usando o método concat()
 var resultado4 = saudacao.concat(", ",nome, ", como vai?");
console.log(resultado3); // Saída: "Olá, Kevin, como vai?"

```


## Number

O `number` inclui tanto números inteiros quanto fracionados.

### Sintaxe


```js
const idade = 25; // inteiro
console.log(idade); // Saída: 25


const preco = 19.99; // fracionado
console.log(preco); // Saída 19.99
```

## Boolean

O `booleano` só pode ter dois valores: `true` ou `false`.<br/>
Eles são comumente usadas em condições para controle de fluxo e loops.

### Sintaxe

```js
let online = true;
console.log(online); // Saída: true

online = false
console.log(online);  // Saída: false
```

## Null

O `null` é um valor que representa "**nenhum valor**" ou "**vazio**".<br/>
Ele deve ser atribuído intencionalmente para indicar que uma variável não tem um valor.

### Sintaxe

```js
let carro = null;
console.log(carro); // Saída: null
```

## Undefined

O `undefined` é o tipo padrão de variáveis que foram declaradas mas nenhum valor foi atribuída a elas.

### Sintaxe

```js
let produto; //A variável declarada
console.log(produto); // Output: undefined
```

<Admonition type="note" title="Nota">
  <h2>Considerações Importantes</h2>

  - **Diferença entre null e undefined**:<br/>
    O `null` é explicitamente atribuído, enquanto `undefined` significa que a variável ainda não foi inicializada.
  
  - **Coerção de Tipo**:<br/>
    JavaScript realiza coerção de tipo automática, por exemplo, ao somar um `number` e uma `string`, o `number` será convertido em `string`.


  - **Valores False**:<br/>
    Em condicionais, além de `false`, os seguintes valores são considerados `false`:<br/>
    `0` (number zero), `""` (string vazia), `null`, `undefined`, e `NaN`.

  - **Verificação de Tipo**:<br/>
    Podemos usar `typeof` para verificar o tipo de uma variável. Exemplo

    ```js
    let value;
    console.log(typeof value); // Saída: "undefined"

    value = 'value';
    console.log(typeof value); // Saída: "string"

    value = 1;
    console.log(typeof value); // Saída: "number"

    value = 1.90;
    console.log(typeof value); // Saída: "number"

    value = true;
    console.log(typeof value); // Saída: "boolean"

    value = null;
    console.log(typeof value); // Saída: "null"
    ```
</Admonition>


## Exercícios Propostos

### Exercício 1: Trabalhando com Strings
Declare uma variável `nome` com o seu nome e outra variável `sobrenome` com seu sobrenome. Em seguida, concatene essas duas variáveis usando o operador `+` e exiba o resultado. Agora, faça o mesmo usando template literals.

<details>
  <summary>Resposta</summary>
```js
  const nome = "Kevin";
const sobrenome = "Medeiros";
console.log(nome + " " + sobrenome); // Usando o operador +. Saída: "Kevin Medeiros"
console.log(`${nome} ${sobrenome}`); // Usando template literals. Saída: "Kevin Medeiros"
```
</details>

### Exercício 2: Diferença entre `null` e `undefined`
Crie uma variável `carro` e atribua a ela o valor `null`. Em seguida, crie uma variável `bicicleta` sem atribuir nenhum valor. Exiba ambas as variáveis no console e analise as saídas. Analise qual é a diferença entre os dois valores.

<details>
  <summary>Resposta</summary>
  ```js
  let carro = null;
let bicicleta;
console.log(carro); // Saída: null
console.log(bicicleta); // Saída: undefined
// `carro` é definido como intencionalmente vazio, enquanto `bicicleta` não recebeu nenhum valor.
```
</details>

### Exercício 3: Trabalhando com Booleanos
Crie uma variável `login` e defina-a como `true` para indicar que o usuário está logado. Em seguida, mude o valor para `false` para indicar que ele fez logout. Exiba o valor de `login` após cada atribuição e reflita sobre o uso de booleanos para representar estados.

<details>
  <summary>Resposta</summary>
  ```js
  let login = true;
console.log(login); // Saída: true

login = false;
console.log(login); // Saída: false
// Booleanos são úteis para indicar estados de "sim/não" ou "ativado/desativado".
```
</details>

### Exercício 4: Operação com Numbers
Declare uma variável `precoProduto` com um valor numérico, por exemplo, 49.99. Em seguida, crie uma variável `quantidade` e atribua um número inteiro a ela. Multiplique `precoProduto` por `quantidade` e exiba o resultado, representando o valor total da compra.

<details>
  <summary>Resposta</summary>
  ```js
  const precoProduto = 49.99;
const quantidade = 3;
const totalCompra = precoProduto * quantidade;
console.log(totalCompra); // Saída: 149.97
```
</details>

### Exercício 5: Verificação de Tipo com `typeof`
Declare uma variável `idade` e atribua um valor numérico a ela. Em seguida, use `typeof` para verificar o tipo de dado da variável. Altere o valor de `idade` para uma string e verifique novamente o tipo de dado. O que acontece?

<details>
  <summary>Resposta</summary>
  ```js
  let idade = 25;
console.log(typeof idade); // Saída: "number"

idade = "25";
console.log(typeof idade); // Saída: "string"
// O tipo de `idade` mudou de `number` para `string` ao mudar o valor.
```
</details>
