## Instalando Node.js

Você pode seguir pelo site official do [**_Node_**](https://nodejs.org/pt/download/package-manager) ou por aqui mesmo.

### Atualize os pacotes do sistema

Antes de instalar qualquer novo software, é uma boa prática garantir que todos os pacotes do sistema estejam atualizados. Abra o terminal e execute os seguintes comandos :

```sh
sudo apt update
sudo apt upgrade
```

### Instale a [_nvm_](https://github.com/nvm-sh/nvm) e o Node

O nvm conhecido como **_Node Version Manager_**, ou **_Gestor de Pacote de Node_**

```sh
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.1/install.sh | bash

# Instale a ultima versão LTS do node
nvm install --lts
# Defina essa versão como global
nvm use global default
```

### Verificar a instalação

Para garantir que o Node.js e o npm foram instalados corretamente, verifique as versões com os seguintes comandos :

```sh
# É pare retornal a versão que foi instalada ex: v20.17.0
node -v


# E aqui ex: 10.8.2
npm -v
```

<Admonition type="info">
  Não se preocupe se sua versão for diferente da que é mostrado no exemplo.
  Provavelmente ela vai ser. Isso acontece porque o node está em constante
  melhora.
</Admonition>

## Instalando VsCode

### Baixe o arquivo

Entre no site do [**_Visual Studio Code_**](https://code.visualstudio.com).
Olhe qual sua distro com comando `lsb_release -a`, pesquise qual executável para sua distro, **.deb** ou **.rpm** e baixe o arquivo.

### Instale o VsCode

- Navegue até a pasta onde o arquivo foi baixado ( geralmente `cd ~/Downloads` )
- Execute o comando para instalar :

```sh
sudo apt install <nome-do-arquivo.deb>
```
